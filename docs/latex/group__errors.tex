Retrieves information about the last error that occurred within the context of libsir or lower. 

\begin{DoxyRemark}{Remarks}
Most C library and OS calls made by libsir are evaluated for failure. If a failure of this type is encountered, this function returns \doxylink{group__errors_gga06fc87d81c62e9abb8790b6e5713c55ba0f293a6020f98a7e41b8afea0e402046}{SIR\+\_\+\+E\+\_\+\+PLATFORM}, and {\ttfamily message} will contain a string identifying the underlying error code and the message as reported by the platform.
\end{DoxyRemark}
