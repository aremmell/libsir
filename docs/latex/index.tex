\doxysection*{Synopsis}

libsir is a cross-\/platform, thread-\/safe logging library written in C that is designed to simplify and streamline the generation and distribution of human-\/readable information in software.

Using libsir, you can make a single call that simultaneously sends information to multiple destinations, {\itshape each with their own individual preference for format and levels}.

Each \textquotesingle{}level\textquotesingle{} or \textquotesingle{}priority\textquotesingle{} of output can be visually styled however you wish for {\ttfamily stdio} destinations. This allows for immediate visual confirmation that a message appeared in the terminal that you should pay attention to, while less pressing information is more subtle in appearance.

\doxysection*{Notables}


\begin{DoxyItemize}
\item No dependencies (other than {\ttfamily libc} and {\ttfamily pthreads}). On Windows, libsir uses the native sychronization API.
\item Accompanied by a robust test suite to ensure dependable behavior, even if you make modifications to the source.
\item Won\textquotesingle{}t fail silently--- C library or platform level errors are captured and stored for posterity, on a per-\/thread basis. The {\itshape function, file, and line number from which it orginated} are also captured. Additionally, libsir defines its own set of error messages to aid in troubleshooting.
\item Hardened--- every function contains sanity checks for internal state as well as arguments passed in through the external interfaces. libsir\textquotesingle{}s make recipes use -- {\itshape Wpedantic}, too.
\item \href{https://aremmell.github.io/libsir}{\texttt{ Full documentation}}, thanks to Doxygen.
\end{DoxyItemize}

\doxysubsection*{Preprocessor}


\begin{DoxyItemize}
\item For my own development purposes, I included a preprocesor define that causes libsir to log certain events to {\ttfamily stderr} that may be useful in diagnosing problems if/when they arise\+: {\ttfamily -\/DSIR\+\_\+\+SELFLOG}.
\item You should also be aware that {\ttfamily assert} is used in libsir. If you build {\itshape without} {\ttfamily -\/DNDEBUG}, and an {\ttfamily assert} triggers, your program will halt ({\ttfamily assert} calls {\ttfamily abort} if the expression is {\ttfamily false}).
\end{DoxyItemize}

\doxysubsection*{Why libsir?}

libsir was originally written in 2003 for my own personal use, and I rediscovered it recently. I looked at it, and realized that it was woefully inadequate and decided to compeltely rewrite it. I also decided to make it publically available, because I think it\textquotesingle{}s a pretty good library, and I hope that it will be useful to others.
