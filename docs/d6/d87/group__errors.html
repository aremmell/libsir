uint16_t sir_geterror ( sirchar_t  message[256])

#include <sir.h>

Retrieves information about the last error that occurred within the context of a library call.

Note
Most C library and OS calls made by SIR are evaluated for failure. If a failure of this type is encountered, this function returns SIR_E_PLATFORM, and message will contain a string identifying the underlying error code and the message as reported by the platform.
