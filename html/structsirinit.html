Detailed Description

Initialization data for the library.

Allocate an instance of this struct and pass it to sir_init in order to begin using the library.

Don't forget to call sir_cleanup when you're done.

Member Data Documentation

const sirchar_t* sirinit::processName

A custom name to include in output (default: none). If SIRO_NONAME is set in the options for the destination, no name is included in output. Set to NULL for the default.

FILE* sirinit::selfOutput

Where SIR should send its own internal logging (default: stderr) to assist in diagnosing any potential issues that may arise. Set to NULL for the default.

Attention
Only available if DEBUG is defined in the preprocessor. Use setlinebuf(3) in order to avoid any delays in output if overridden. SIR does not call fclose(3) on the stream upon cleanup.
