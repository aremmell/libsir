Enumeration Type Documentation

anonymous enum

Logging levels

Each enum value corresponds to a function that sends output to all destinations for that level (sirdebug and pals).

Since the values are also flags (i.e.: powers of two), they can be combined to filter different levels of importance but not strictly linearly. Any combination of levels can be used to configure an output destination.

Enumerator

Nuclear war, Armageddon, etc.

SIRL_INFO 

Informational messages.

SIRL_DEBUG 

Verbose debugging output.

SIRL_ALL 

Includes all logging levels.
