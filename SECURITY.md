# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please [create a new security vulnurability issue](https://github.com/aremmell/libsir/security/advisories/new) and I will review it as quickly as I can.
